# Derek Gygax Porfolio

> Portfolio for Derek Gygax

## Prerequisites

- Node 18.8.0+
- npm 8.18.0+

## Install/Use Node 18

```
nvm install 18
nvm use 18
```

## Install Dependencies
### Need to do the extra install next-intl@3.0.0-beta.19 because it is still in beta
```
npm install
```

## Build Project

```
npm run build
```

## Launch built project. After having built

```
npm start
```

## Run Dev Server

```
npm run dev
```
